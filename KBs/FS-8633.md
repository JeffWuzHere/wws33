# Knowledge Base Entry: Using Wholesale Suite on a WordPress Multisite Network

## 🎯 Executive Summary
**The Issue:** Customer was transitioning to a WordPress multisite environment and needed to know if Wholesale Suite requires special configuration and what license type is needed.
**The Fix:** No special configuration is required beyond installing the plugin at the network level; however, an unlimited site license is mandatory — single-site licenses will not work correctly across a multisite network.

---

## 🛠 Final Resolution (The "2nd Me" Guide)
> [!TIP]
> **Use this solution if a customer asks about:** multisite, wordpress network, multiple sites, network activation, multisite license, network install wholesale suite
>
> **Steps to solve:**
> 1. Confirm the customer has (or needs to purchase) an **unlimited site license** for Wholesale Suite. Single-site or multi-site (limited) licenses will not function correctly on a WordPress multisite network.
> 2. Install and network-activate the Wholesale Suite plugin(s) from the Network Admin dashboard (Network Admin > Plugins > Network Activate).
> 3. Once network-activated and the license is confirmed, configure Wholesale Suite settings on each sub-site as normal — no special network-level configuration is needed beyond the activation step.
> 4. Refer the customer to: https://wholesalesuiteplugin.com/kb/how-to-install-and-activate-the-plugins-for-multisite/

## 📝 Sanitized Interaction History

We're transitioning to a multisite environment. Are there any specific steps, configurations, or limitations to be aware of when using Wholesale Suite across multiple sites?

**[SUPPORT_AGENT]:**
No special configuration is needed to make Wholesale Suite work on a multisite network. However, you will need an unlimited site license — single-site licenses will not work as intended. Once the license is active and the plugin is installed at the network level, you can configure the wholesale settings per site as normal. See: https://wholesalesuiteplugin.com/kb/how-to-install-and-activate-the-plugins-for-multisite/
