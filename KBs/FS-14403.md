# Knowledge Base Entry: Reorder Quantity Discount Table to Display After Product Short Description

## 🎯 Executive Summary
**The Issue:** By default, the Wholesale Prices Premium quantity-based discount table renders before the product short description on single product pages, and there is no built-in setting to change this position.
**The Fix:** Add a custom PHP snippet (via child theme `functions.php` or a code snippet plugin) that moves the quantity discount table to render after the short description.

---

## 🛠 Final Resolution (The "2nd Me" Guide)
> [!TIP]
> **Use this solution if a customer asks about:** quantity discount table position, move discount table, reorder pricing table product page, discount table after description, wholesale pricing table location
>
> **Steps to solve:**
> 1. Take a full site backup before proceeding.
> 2. Install a code snippet plugin such as WPCode (https://wordpress.org/plugins/insert-headers-and-footers/) if a child theme is not available.
> 3. Add the custom snippet from https://pastebin.com/h79UcMZc to either:
>    - Your child theme's `functions.php`, or
>    - The WPCode plugin's snippet manager.
> 4. Save and verify on the frontend that the quantity discount table now appears below the product short description.
> 5. Test thoroughly to confirm no layout breakage.

## 💡 Internal Insights (If applicable)
- There is no built-in setting in Wholesale Prices Premium to control the render position of the quantity discount table. This is a common customization request.
- The snippet hooks into WooCommerce's product page action hooks to unhook the table from its default position and re-hook it after the short description output.
- Always recommend WPCode (formerly Insert Headers and Footers) as the snippet manager for customers without a child theme — it's safer than editing `functions.php` directly on the active theme.
