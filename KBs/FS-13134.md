# Knowledge Base Entry: Bulk Excluding Products from Wholesale Order Form

## 🎯 Executive Summary
**The Issue:** A store with 1,300+ products needs to show only ~100 wholesale products on the Order Form, but there is no native bulk-exclude feature — excluding 1,200 products one by one is not feasible.
**The Fix:** Two native approaches avoid per-product exclusion: (1) assign wholesale products to a dedicated category and configure the Order Form to show only that category, or (2) enable the "Wholesale Products Only" visibility option so the form automatically shows only products that have a wholesale price set.

---

## 🛠 Final Resolution (The "2nd Me" Guide)
> [!TIP]
> **Use this solution if a customer asks about:** bulk exclude products order form, too many products to exclude one by one, only show certain products wholesale order form, filter products on order form, wholesale only products, category filter order form, 1000 products exclude
>
> **Steps to solve:**
> 1. **Option A — Category-based filtering (recommended for partial catalogs):**
>    - Create a dedicated WooCommerce category (e.g., "Wholesale Products") and assign only the ~100 wholesale-eligible products to it.
>    - In the Order Form settings, set the category filter to display only that category. Customers browsing the form will only see those products.
>
> 2. **Option B — "Wholesale Products Only" visibility setting:**
>    - Navigate to Wholesale > Settings and enable the option that restricts the Order Form/shop pages to only display products that have a wholesale price assigned.
>    - Reference: https://wholesalesuiteplugin.com/kb/how-to-adjust-visibility-of-wholesale-products/
>    - Set wholesale prices only on the ~100 products that should appear. Products without a wholesale price will be hidden from wholesale customers automatically.
>
> 3. **Bulk-setting wholesale prices via CSV (to support Option B):**
>    - Use the WooCommerce import/export tool with Wholesale Suite's import extension to bulk-assign wholesale prices.
>    - Reference: https://wholesalesuiteplugin.com/kb/how-to-import-wholesale-data-into-your-woocommerce-store/
>
> 4. **Not currently available:** Bulk CSV import of per-product Order Form include/exclude flags. A feature request exists for this.

## 📝 Sanitized Interaction History

**[CUSTOMER]:**
I have a client with 1,300+ products but only ~100 will be wholesaled. Excluding 1,200 products one by one from the Order Form is not possible. Is there a bulk CSV import option to set the Order Form exclusion flag, similar to how Wholesale Prices Premium allows bulk exclusion via CSV?

**[SUPPORT_AGENT]:**
There is no bulk CSV import for Order Form include/exclude settings at this time. However, you can avoid the problem entirely with two approaches:
- Set a dedicated category for the wholesale products and configure the Order Form to show only that category.
- Enable "Wholesale Products Only" mode so the form only shows products that already have a wholesale price. You would then use the bulk import tool to assign wholesale prices to only the ~100 relevant products.

A feature request has been raised to support bulk upload of Order Form product settings via CSV.

## 💡 Internal Insights (If applicable)
- Option B (Wholesale Products Only) is the cleanest solution when the number of wholesale-eligible products is small relative to the catalog — no manual exclusion needed at all.
- The bulk import/export tool handles wholesale prices and other product-level data but does not yet cover per-product Order Form visibility flags.
- Feature request tracked at: https://github.com/Rymera-Web-Co/woocommerce-wholesale-order-form/issues/108
