# Knowledge Base Entry: International Shipping Not Showing for Wholesale Customers — Empty Table Rate Rules

## 🎯 Executive Summary
**The Issue:** International (non-UK) wholesale customers had no shipping options available at checkout because the wholesale-specific Table Rate Shipping method in the "Outside UK" shipping zone had been configured with a wholesale restriction but contained no actual shipping rules/rates, causing wholesale customers in those zones to see zero shipping options.
**The Fix:** Add the required shipping rates/rules to the wholesale-specific Table Rate Shipping method within each affected shipping zone; ensure the Wholesale > Settings > Shipping restriction points to the correct wholesale-specific Table Rate method (not the general one).

---

## 🛠 Final Resolution (The "2nd Me" Guide)
> [!TIP]
> **Use this solution if a customer asks about:** international shipping not showing wholesale, wholesale customers no shipping options, shipping missing for wholesale, outside UK shipping wholesale, table rate shipping wholesale empty, wholesale shipping zone not working, non-UK wholesale no checkout shipping
>
> **Steps to solve:**
> 1. Go to WooCommerce > Settings > Shipping and review all shipping zones that cover the affected countries/regions.
> 2. In each affected zone, check whether there is a wholesale-specific Table Rate Shipping method (e.g., "Table Rate Shipping - Wholesale") separate from the standard Table Rate method.
> 3. Click into the wholesale-specific Table Rate method and confirm whether it has any shipping rules/rates defined. An empty rule set means wholesale customers matched to that zone will have no shipping options — this is the most common root cause.
> 4. Add the required shipping rates to the wholesale-specific Table Rate method in each affected zone.
> 5. Go to Wholesale > Settings > Shipping and confirm the restriction is mapped to the wholesale-specific Table Rate method (not the general one). If two Table Rate methods exist in the zone, it is easy to accidentally restrict the wrong one.
> 6. Test by adding items to the cart as a wholesale user and simulating checkout from an international address — confirm shipping options now appear.
> 7. When creating multiple shipping zones, ensure countries do not overlap between zones. If a country appears in two zones, WooCommerce uses the highest-priority zone, which may cause unexpected results.

## 💡 Internal Insights (If applicable)
- The most overlooked cause of "wholesale customers see no shipping options" is that a wholesale-restricted Table Rate method exists in the zone but has no rates/rules inside it. WooCommerce will show an empty shipping selection for those customers.
- Always verify both: (1) the correct Table Rate method is referenced in Wholesale > Settings > Shipping, AND (2) that method contains actual shipping rules.
- When multiple Table Rate methods exist in the same zone (common when setting up wholesale-specific rates), naming them distinctly (e.g., "Table Rate - General" vs "Table Rate - Wholesale") prevents accidental mis-mapping.
- Overlapping shipping zones cause WooCommerce to use the highest-priority zone, potentially hiding shipping methods. Keep country lists in each zone mutually exclusive.
- For differential pricing across regions (e.g., Europe vs. US/ROW), separate shipping zones per region are needed; countries can be grouped within a single zone's Table Rate rules using WooCommerce Table Rate's built-in row conditions.
